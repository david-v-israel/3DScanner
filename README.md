Java‑based application to automate the capture and validation of serial numbers and MAC addresses from QR barcodes on telecommunications devices. Implemented logic to process, sort, and store scanned data directly into the company database, improving accuracy and workflow efficiency. Designed the system to significantly reduce manual scanning time and minimize human‑error‑related data capture issues.
